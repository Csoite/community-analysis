## 浏览器常用插件
# Microsoft Edge 扩展
- Translator For Microsoft Edge 
-通过 Translator for Microsoft Edge，你可以翻译 60 多种语言的外语网页和文本选择。在访问外语网页时，请在地址栏中查找 Microsoft Translator 图标。单击该图标可将网页即时翻译为你的首选语言。你还可以选择要翻译的文本，然后右键单击以获取行内翻译。
- Adblock Plus (Beta)/Adblock：
-Adblock Plus主要功能就是拦截过滤网页广告，包括视频广告、横幅广告、弹窗广告和其他广告。其他功能有网页提速、防网络跟踪、禁止恶意域名、可接受广告等。
- Tampermonkey 
-该扩展程序基本上允许用户更改网页上的Javascript以启用不同的功能
- Office浏览器扩展 
-无论你是否在该特定计算机上安装了Office，此扩展都将Microsoft Office的功能带到Edge浏览器。它将图标放置在Edge工具栏中，该图标允许访问存储在OneDrive和设备本地存储中的MS Office文件。该扩展程序支持Word，Excel，PowerPoint和OneNote，并适用于个人和企业帐户。你可以直接在浏览器中操作这些文件，从而使工作效率更高，而不会浪费系统资源或在不同应用之间来回跳转。
- OneNote Web Clippe 
-将网页的剪裁直接带到你的OneNote帐户中，无论是整页还是其中的一部分。然后，你可以在OneNote应用程序中执行所有常规注释，共享等操作。

# FireFox 扩展
- uBlock Origin
-一款高效的网络请求过滤工具，占用极低的内存和 CPU。
- To Google Translate 谷歌快译
-右键单击一段文本，点击它旁边的“翻译”图标，即可翻译或收听它的发音。
- Search by Image
- Download Manager (S3)
- Worldwide Radio
